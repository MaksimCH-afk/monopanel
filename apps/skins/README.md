# skins — Brandskins

| | |
|---|---|
| **Назначение** | Шаблоны и темы брендов |
| **Стек** | Flask + gunicorn |
| **Порт** | **3336** (исходный был 8888) |
| **Данные** | файлы (состояние — в `instance/`) |
| **Авторизация** | нет |

## Куда положить код

Исходники Flask-приложения — в `apps/skins/`. Точка входа ожидается как
`app:app` (модуль `app.py`, объект `app`). `requirements.txt` — в корень папки;
зависимости ставятся во venv `/venv/skins` (Flask 3.1).

## Маршруты

- `/`       — публичный сайт
- `/admin`  — панель (плитка дашборда ведёт сюда)

## Состояние

Выбранные темы хранятся в файлах внутри `apps/skins/instance/`. Каталог
примонтирован томом в `docker-compose.yml`, чтобы состояние переживало
пересборку образа.

## Запуск (см. supervisord.conf)

`/venv/skins/bin/gunicorn -b 0.0.0.0:3336 app:app`
