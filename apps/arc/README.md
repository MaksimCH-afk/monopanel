# arc — Web Archive Checker

| | |
|---|---|
| **Назначение** | Прогоны доменов по веб-архиву |
| **Стек** | FastAPI (async) |
| **Порт** | **3335** (исходный был 8000) |
| **БД** | SQLite / PostgreSQL |
| **Авторизация** | нет |

## Куда положить код

Исходники FastAPI-приложения — в `apps/arc/`. Точка входа ожидается как
`main:app`. `requirements.txt` — в корень папки; зависимости ставятся во
venv `/venv/arc`.

## Данные

Стартует с чистого листа — перенос данных для arc не выполняется.
Если используется PostgreSQL, параметры подключения задаются через `.env`.

## Запуск (см. supervisord.conf)

`/venv/arc/bin/uvicorn main:app --host 0.0.0.0 --port 3335`
